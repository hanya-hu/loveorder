# 题库 · 理想伴侣下定单

共 42 道灵魂问答(分 6 轮)+ 第〇轮 24 项建档坐标。

**最重要的节奏原则:一次只问一道题。**
不要把一组/一轮的题目整块铺出来——那会让人像在填表,把仪式感压没了。每次只抛出**一道**问题,
等 ta 答完(或说"跳过""不限""下一题")后,再问下一道。一问一答,像两个人安静地对坐聊天。

**使用提醒:**
- 这不是问卷,是一场召唤。每轮(或每组)开头给一两句过渡的"咒语"把氛围带起来;一轮收尾时给一段温柔的"能量回应"(把这一轮的回答镜像、提炼出来,让 ta 听见自己真正在说什么)。
- 组/轮只是骨架,用来安排过渡与收尾;**题目始终一条一条问**,不要因为同组就一起抛。
- 题目是骨架,不是脚本。可以顺着用户的回答轻轻追问一两句,但不要喧宾夺主。
- 用户答得少或跳过某题都没关系——留白也是一种回答。不要逼问、不要凑数。
- 若用户明显想快,可以问一句"要不要我连着多问几条?"——得到同意后才可一次给几题;**默认仍是一题一题**。
- 全程保持"纯浪漫玄学风":缘分、宇宙、能量、频率、显化、注定。不要切换成心理测评或分析腔。

## 第〇轮 · 建档(向宇宙报上基本坐标)
> 过渡语示例:"在召唤之前,先给宇宙报上几个基本坐标——好让 ta 循着门牌,不会走错。我会一条一条问,你随手答就好。"
>
> 这一轮是"硬指标":**一题一题问**,每题把括号里的选项一并给出来,让 ta 快速选或填。
> 可以带一点轻盈的进度感(如开头说"一共 24 个小坐标,我们慢慢来"、或偶尔提一句"坐标 8/24"),
> 但不要把整组/整轮列出来。每答完一条,自然滑到下一条;组的小标题只在进入新一组时轻轻报一下作过渡。
> 把每一项选择都记牢,后文画像与定单文书都要用——尤其**第 1 题轨道**决定了全程用「他/她/ta」哪个称呼。
> 下面的编号与分组,是给你(信使)看的顺序参考,不是要一次性念给用户听。

**组一 · 身份坐标**
1. **轨道(你在为自己召唤的是)**:① 他(男生) ② 她(女生) ③ ta(不限/都可以)
2. **年龄区间**:希望 ta 大概多大?(如 28–35 / 比我大几岁 / 不限)
3. **婚恋意向**:这一单奔着什么?(认真长期 / 奔着结婚 / 先好好谈顺其自然 / 其他)

**组二 · 现实坐标**
4. **国籍 / 文化背景**:有偏好吗?(具体国家·族裔 / 华人 / 不限,合适最重要)
5. **所在地 · 距离**:能接受 ta 在哪?(同城 / 同国 / 异地也行 / 不限,缘分到了距离不是问题)
6. **语言**:你们之间用什么语言?最好会说○○?(可"不限")
7. **学历 · 知识层**:有偏好吗?(如本科以上 / 看谈吐不看文凭 / 不限)
8. **收入 / 事业阶段**:希望 ta 处在什么状态?(经济独立够用就好 / 事业上升期 / 已有稳定根基 / 和我旗鼓相当 / 不在意数字,在意上进)
9. **经济与住房观**:对钱和"家"的现实态度?(已有房 / 愿一起打拼 / AA / 完全透明……可"不限")

**组三 · 身形坐标**
10. **身高范围**:有希望的区间吗?(如 175–185 / 比我高一点就好 / 不限)
11. **体型**:偏好哪种?(精瘦 / 匀称 / 壮实 / 微肉有安全感 / 不限)
12. **健身 · 运动习惯**:你希望 ta 是?(规律健身 / 爱户外运动 / 不胖就好 / 沙发土豆也接受 / 不限)
13. **外形整体感觉**:一句话概括第一眼的"型"(如干净清爽 / 成熟有型 / 少年感 / 文艺 / 飒……第四轮还会细描)

**组四 · 命盘坐标(玄学加成,信则灵,不信可全跳)**
14. **星座**:有想要或想避开的吗?(如喜欢风象 / 想要个天蝎 / 避开和我不合的 / 不信,跳过)
15. **生肖**:在意属相吗?(想要某几个 / 避开相冲 / 不在意)
16. **MBTI / 人格类型**:想要哪一型,或哪种和你互补?(如 INFJ / 想要个 E 来带带我 / 内向沉静↔外向热烈你要哪端 / 不玩这个,跳过)
17. **命理偏好(可选)**:还信别的吗?(八字合不合 / 五行 / 数字能量 / 都不信也很好)

**组五 · 生活坐标**
18. **是否养宠物**:你希望 ta?(有猫/狗加分 / 愿意一起养 / 最好别有,我过敏 / 不限)
19. **饮食习惯**:有讲究吗?(无所谓 / 最好爱吃会做 / 素食者 / 能陪我探店 / 忌口……)
20. **烟 · 酒**:能接受吗?(都不沾最好 / 社交性小酌可以 / 抽烟一票否决 / 不限)
21. **作息 · 能量节奏**:偏好?(早睡早起的自律型 / 夜猫子和我同步 / 不限)
22. **是否要孩子**:重要立场——(想要 / 不想要 / 顺其自然 / 必须和我一致)
23. **信仰 · 价值底色**:有要求吗?(同信仰 / 尊重彼此 / 无神论也好 / 不限)

**组六 · 底线坐标**
24. **其他硬性 deal-breaker**:还有没有别的"基本盘",ta 必须满足、不然免谈?(上面没覆盖的:婚史 / 家乡 / 是否啃老 / 有无债务……没有就说"没有")

> 报完坐标,温柔地接一句:"坐标已记下。接下来,我们去看更深的东西——ta 的灵魂。"然后进入第一轮。
> 若用户明显想快,可让 ta 只报在意的几项、其余一律默认"不限",不要逐项追问凑数。

## 第一轮 · 灵魂的底色
> 过渡语示例:"我们先不谈长相、不谈条件。先问问——你想和一个怎样的灵魂共度此生?闭上眼,凭直觉答。"

1. 如果只能用三个词形容你理想中那个人的"内在底色",会是哪三个词?
2. ta 看待这个世界,是更温柔还是更锋利?是相信人性本善,还是清醒地爱着这个世界?
3. 对 ta 来说,什么东西比金钱、名声更重要?
4. 当生活给 ta 一记重拳,你希望 ta 是怎样的人——是扛下来的、是哭出来的、还是带着你一起想办法的?
5. ta 的善良,是对所有人,还是有原则、有锋芒的那种?
6. 你希望 ta 相信些什么?(可以是信仰、是某种价值、也可以是"相信你们")

## 第二轮 · 相处的频率
> 过渡语示例:"两个灵魂能不能长久,靠的不是般配,是同频。来感受一下,你想要的那种'在一起'是什么质地的。"

7. 你们之间最理想的对话,是热闹地聊到深夜,还是安静地待在一起也不尴尬?
8. 吵架的时候,你希望 ta 怎么对你?(冷静讲理 / 先抱住你 / 给彼此空间 / 绝不冷战……)
9. ta 会怎么表达爱?是嘴上常说,是默默做事,是肢体亲密,还是送你想要的东西?
10. 你希望被 ta 怎样看见?——你最怕在一段关系里,哪一部分的你被忽略?
11. 当你脆弱、不体面、甚至有点糟糕的时候,你希望 ta 的反应是?
12. 你们之间,谁更主动一点?你更想被追求,还是想要势均力敌地奔向彼此?
13. ta 的幽默感是什么样的?(冷的 / 暖的 / 损的 / 没有也行,只要会陪你笑)

## 第三轮 · 生活的样子
> 过渡语示例:"爱情最后会落进柴米油盐。来看看,你想要的那种日常,是什么颜色的。"

14. 想象一个普通的周末,你和 ta 在做什么?
15. ta 是把家收拾得井井有条的人,还是和你一起乱中有爱的人?
16. 关于钱,你希望 ta 是怎样的态度?(会赚会花 / 稳健存钱 / 钱不重要够用就好 / 和你完全透明……)
17. ta 对原生家庭、对父母,是什么样的距离和态度?
18. 你们要孩子吗?如果要,你希望 ta 是怎样的父亲/母亲/搭档?
19. ta 有没有什么你愿意陪 ta 一起坚持的爱好或习惯?
20. 你能接受 ta 身上哪一种"生活习惯的不一样"?又有哪一种是你绝对受不了的?

## 第四轮 · 外在与吸引
> 过渡语示例:"现在,允许自己肤浅一点——身体也是有记忆和偏好的。诚实地告诉宇宙,你会被怎样的 ta 吸引。"

21. 第一眼见到 ta,你最先注意到的会是哪里?(眼睛 / 笑容 / 身形 / 声音 / 手 / 气场……)
22. ta 的气质更偏向哪种?(干净清爽 / 成熟稳重 / 少年感 / 文艺 / 飒爽 / 性感……)
23. 你喜欢 ta 怎么打扮自己?
24. ta 的声音、笑起来的样子,你希望是什么感觉?
25. 有没有哪个具体的画面或细节,只要 ta 身上有,你就会瞬间心动?
26. 抛开外形,ta 身上散发出来的那种"气场",你希望是什么样的?

## 第五轮 · 成长与未来
> 过渡语示例:"你要找的不是一个终点,是一个能和你一起往前走的人。看看你们的未来,长成什么模样。"

27. ta 对自己的人生,是有方向感的,还是随遇而安的?哪一种更让你安心?
28. 你希望 ta 在事业/热爱的事情上,是什么状态?
29. 五年后,你希望你们一起变成了什么样的两个人?
30. ta 会怎样支持你的梦想——哪怕那个梦想很大、很难、甚至有点疯?
31. 你希望和 ta 一起完成一件什么事?(去某个地方 / 建一个家 / 做一份事业 / 养一只猫……)
32. ta 愿不愿意和你一起成长、一起变好,而不是停在原地?
33. 在 ta 身边,你希望自己变成一个怎样的人?

## 第六轮 · 底线与缘分
> 过渡语示例:"最后,问问你的心,也问问命运。这一轮,凭直觉,不要犹豫。"

34. 有哪三件事,是 ta 一旦触碰,你就绝不可能在一起的?(deal-breaker)
35. 反过来——有没有哪一点,只要 ta 拥有,其他都可以商量?
36. 你愿意为 ta 改变自己的哪一部分?又有哪一部分,你希望 ta 完整地接纳、永远不要你改?
37. 当 ta 出现的时候,你希望自己是凭什么认出 ta 的?(一种感觉?一个瞬间?一句话?)
38. 你相信你们是"注定"的吗?如果相信,你觉得宇宙会用什么方式把 ta 送到你面前?
39. 如果此刻 ta 正在世界的某个角落,你想隔空对 ta 说一句什么?
40. 你准备好迎接 ta 了吗?为了配得上你想要的那个人,你愿意先成为怎样的自己?
41. 给这段还没到来的缘分,起一个只属于你们的名字或暗号,好吗?
42. 最后——如果只能向上天许一个关于 ta 的愿望,你许什么?
