# Loveorder · 向上天下一张定单 ✦

一个有仪式感的 **Claude Code Skill**：通过约 40 道分主题的访谈式问题，带你一步步看清自己理想伴侣的画像，最后生成一封精美的、可亲手「发送给宇宙」并保存收藏的**《致宇宙的定单》**。

> 你不是在"挑剔"或"列条件"，你是在**显化**。
> 当你足够清楚自己要什么、并真心相信 ta 会来，宇宙就会开始安排。
> 看清楚，本身就是召唤的第一步。

纯浪漫玄学风：缘分、宇宙、能量、频率、显化、正缘。外壳是仪式感与松弛感，内核是一次诚实的自我看见。

---

## 体验流程（约 20–30 分钟）

| 幕 | 发生什么 |
|---|---|
| 一 · 开场仪式 | 把你带进状态，预告时长，邀请你找个安静时刻 |
| 二 · 建档 | 向宇宙报「基本坐标」——24 项、六组：身份 / 现实 / 身形 / 命盘 / 生活 / 底线 |
| 三 · 六轮问答 | 灵魂底色 → 相处频率 → 生活样子 → 外在吸引 → 成长未来 → 底线缘分，每轮给「能量回应」 |
| 四 · 画像合成 | 把所有回答**编织成一个正在走来的人**，允许你修正定稿 |
| 五 · 下定单仪式 | 带你做一个「签字画押」的小动作 |
| 六 · 生成文书 | 产出一封 HTML 定单：亲手按下「向宇宙发送此单」→ 仙女棒火花划过 + 「宇宙已签收」印章 → 保存 / 存为 PDF |

### 多轨制
建档第一项就是**轨道**：你在为自己召唤的是 **他（男生）/ 她（女生）/ ta（不限）**。选定后全程称呼与语感自动跟随。

## 安装

> 技能内部名为 `loveorder`（全小写，符合 skill 命名规范）；下面的目录名也请用小写。

**方式一 · 直接放进 skills 目录**
```bash
git clone https://github.com/hanya-hu/Loveorder.git ~/.claude/skills/loveorder
```

**方式二 · 安装打包文件**
下载本仓库（或 [Releases](https://github.com/hanya-hu/Loveorder/releases)）里的 [`loveorder.skill`](loveorder.skill)，在 Claude Code 里安装即可。

装好后，用 `/loveorder`，或对 Claude 说「帮我向宇宙下个恋爱定单」「显化我的正缘」「帮我画画我的理想型」之类，就会触发。
